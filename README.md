# MLP-and-KAN-Data-for-Thermophysical-Properties-of-Diamond-Al-Composites
The repository is data and code of MLP and KAN Data for Thermophysical Properties of Diamond/Al Composites.

#data
The dataset is a CSV file in the “data” folder.

#code 

The code related to MLP and KAN in the paper is provided as an .ipynb file, located in the .ipynb_checkpoints folder, named "MLP vs KAN for diamond-al.ipynb".

And the Python scripts developed for generating the RVE-FE models used in this work will be made publicly available in the same repository upon acceptance of the paper.
